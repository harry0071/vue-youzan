# 有赞商城

>vue重构有赞商城

## 使用方法
将项目clone到本地。然后执行下面的命令

``` bash
# 首先安装依赖
npm i

# 然后运行项目
npm start

# 打包
npm run build
```

## 技术栈
vue + vue-cli + axios + mint-ui + es6

## 插件
- qs：处理查询参数，将字符串格式的查询参数转变成键值对格式
- swiper：轮播图插件
